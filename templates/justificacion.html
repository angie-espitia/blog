Justificación Social

En el entorno social actual la tecnología ha tomado un papel principal, pero en el sector educativo aún existen ciertas dificultades que impiden el uso de estas herramientas y hacen que impartir las clases requiera de un contacto frecuente con el docente.

Es por esto que con este proyecto se busca dar una solución óptima a esta problemática por medio de un recurso educativo realizado en un entorno WEB, teniendo por objetivo que los estudiantes aprovechen el uso de las TIC para la realización de actividades en pro de su formación y, además, el docente tenga un apoyo seguro para la orientación de sus actividades.

De esta manera, los padres de familia y/o acudientes podrán estar seguros de que la educación orientada en la institución es de una excelente calidad y garantiza el desarrollo en el aprendizaje.

Además, las directivas de la institución se verán beneficiadas con los resultados obtenidos, y con ello, aumentando la demanda por parte de personas particulares que se interesen en este recurso educativo.

Justificación Académica

El proyecto es creado en la Universidad Piloto de Colombia en la carrera de Tecnología en Desarrollo de Sistemas de Información y, con el cual, se busca satisfacer una necesidad que se viene presentando en una comunidad estudiantil a la hora de impartir las clases de inglés ofreciendo un soporte adicional para que así se garantice un mejor aprendizaje.

Justificación Técnica

En la realización de este proyecto se usaran varios de los conocimientos adquiridos a lo largo de la carrera Tecnológica:

La interfaz gráfica se realizará bajo el lenguaje de HTML y CSS para el diseño, lenguajes de programación como PHP y JavaScript para la interacción de las páginas con el usuario de una forma más dinámica. También se hace uso de un framework para diseño de aplicaciones WEB como lo es Bootstrap, el cual contiene plantillas con tipografía, formularios, botones, cuadros, menús de navegación y otros elementos para el diseño de las páginas.

Para la base de datos se empleará MySQL.

Además, se usará otro framework como Django para comunicar la base de datos con la interfaz gráfica haciendo uso de Python, un lenguaje de programación bastante flexible ideal para el desarrollo de aplicaciones en entorno WEB.
